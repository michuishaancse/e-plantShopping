# e-plantShopping

## Project Overview

e-plantShopping is an online plant shopping application that allows users to browse a variety of plants, view plant details, and add plants to their shopping cart. The application provides a simple and user-friendly interface for exploring and purchasing plants online.

### Features

* Browse different categories of plants
* View plant information and details
* Add plants to the shopping cart
* Manage cart items
* Responsive and intuitive user interface

### Technologies Used

* React
* JavaScript
* HTML/CSS
